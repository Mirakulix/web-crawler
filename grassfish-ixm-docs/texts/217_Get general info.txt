If the player is not in kiosk mode , you can see the info window. It contains the most important information about the system and the player hardware. For example, a playerâ€™s IP address or the version of the operating system. Note Without user interaction, this window closes after 60 seconds and the player resumes playing spots. If you enter a password, every character counts as an interaction which keeps the window open for an additional 60 seconds. Access the menu To access the menu for options and settings , enter the current password at the top of the info window and select UNLOCK .
