Titel: Disable F keys
URL: https://docs.grassfish.com/docs/disable-f-keys
Zeitpunkt: 2025-07-17 18:42:20
==================================================

This article describes how to enable or disable the F key functions of physical keyboards. This affects the overlays F2, F3, F8, and the F10 and F12 functions. System requirements Ensure that you meet the following system requirements: Android Player version 11.8.9-coba or 11.8.7 Server version 11.8.x or later Android Player configurations updated on server and IXM One Disable F keys By default, F key functions are enabled in the player settings and in IXM One configuration group. You can, however, disable them. Configure the player To disable the F key functions in the player settings, perform the following steps: On your device, go to Common . Disable the setting Enable F-Key Functions . Configure the IXM One To disable the F key functions in IXM One, perform the following steps: In IXM One, go to Administration > Player . Select the Configuration groups tab. In the Configuration groups pane, double-click on the configuration group. Alternatively, select the configuration group and click on Edit . In the Modules pane, select Common . Disable the setting Enable F-Key Functions . Click Save to save you changes. The settings are synchronized between the player and IXM One.
