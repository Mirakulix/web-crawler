Titel: HTML Composer spot import fails
URL: https://docs.grassfish.com/docs/composer-spot-import-fails
Zeitpunkt: 2025-07-17 18:30:35
==================================================

Symptoms You get an error message when you try to import exported HTML Composer spots. Resolution The following steps can help you identify and solve the issue: Ensure that the fonts that are used in the spot are available. Download the fonts from the old server and upload them to the new server. Note For more information, see our article on customizing fonts .
