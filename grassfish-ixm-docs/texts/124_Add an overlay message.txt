Titel: Add an overlay message
URL: https://docs.grassfish.com/docs/add-an-overlay-message
Zeitpunkt: 2025-07-17 18:39:05
==================================================

You can add an overlay with a message on the Android Player. This can be useful, for example, to inform customers that they should touch the screen for purchases. To add an overlay message, perform the following steps: Open the player configuration or the configuration group: For an individual player, go to Channels > Location . Double-click on your player and select Edit next to Configuration in the Edit player window. For a configuration group, go to Administration > Player > Configuration groups and double-click on the configuration group. In the Edit configuration group window, select Display extended modules . Under Modules , select Player > Appearance . Under Settings , select Enable overlay message . Configure the following settings: Overlay message text : Enter the text that you want to display. Overlay message text size : Specify the size of the message text in pixels. Overlay message position : Select whether the message is positioned at the Top or Bottom of the screen. Click Save to save your changes. Restart the player service or reboot the player for your changes to take effect.
